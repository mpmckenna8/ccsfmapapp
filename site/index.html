A simple, responsive map web app for finding things related to CCSF. It makes use of: Bootstrap 3, Leaflet, and typeahead.js. Open source, MIT licensed, and available on GitHub.

Features
  • Data for buidlings at some campuses
  • jQuery loading of external GeoJSON files
  • Using open data from OpenStreetMap and compiled data.
  • Ability to search for features.

The data provided on this site is for informational and planning purposes only.

Absolutely no accuracy or completeness guarantee is implied or intended. All information on this map is subject to such variations and corrections as might result from a complete title search and/or accurate field survey.

Basemap data and much building and reference data derived from OpenStreetMap

All of the data not on the basemap can be found at the Github repository for this project.
